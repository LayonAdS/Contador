# 📊 Contador com Mudança de Cores

Um contador simples que altera a *cor dos números conforme o valor aumenta*.
Este projeto foi criado como exercício/prática de desenvolvimento e manipulação de estado e estilos dinâmicos.

## 🚀 Funcionalidades

* Incrementar o valor do contador
* Exibir o valor atual
* Alterar a *cor do número automaticamente conforme ele aumenta*
* Interface simples e fácil de entender

Exemplo de comportamento:

| Valor   | Cor      |
| ------- | -------- |
| 0 - 9   | Preto    |
| 10 - 19 | Vermelho |
| 20+     | Azul     |


## 🛠️ Tecnologias utilizadas

* Android Studeo

## 🎯 Objetivo do projeto

Este projeto tem como objetivo:

* Praticar lógica de programação
* Trabalhar com manipulação do DOM
* Implementar mudanças visuais baseadas em estado
